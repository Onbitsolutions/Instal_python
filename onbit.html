🚀 Python Setup Guide for Beginners

Everything you need to start coding — step by step.

1️⃣ Install Python 3

Get the latest version of Python (it’s free and open source). Remember to tick "Add Python to PATH" when installing on Windows.

Download Python 3

2️⃣ Install PyCharm (IDE)

PyCharm makes coding in Python easier. Use the free Community Edition for beginners, or the Professional version if you prefer more features.

Get PyCharm

3️⃣ Optional: One-Click Data Science Setup

If you’ll be working with data, you can install Anaconda. It includes Python + popular libraries (NumPy, Pandas, Matplotlib, Jupyter, etc.).

Download Anaconda

4️⃣ Must-Have Python Libraries

After installing Python, open your terminal or PyCharm terminal and run these commands:

pip install requests numpy pandas matplotlib flask pytest

These tools will help you with web apps, data science, and testing.

💡 Extra Tools (Optional)

  • Git — for version control
  • VS Code — lightweight alternative to PyCharm
  • GitHub — store your projects online
